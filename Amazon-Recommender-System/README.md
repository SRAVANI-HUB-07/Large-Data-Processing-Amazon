# Amazon Recommender System

A comprehensive data analytics engine and recommender system built on Apache Spark for Amazon product data.

## Features

### Search Engine
- **Best Sellers**: Top N products by sales rank in any category
- **Review Analysis**: Products filtered by review count and rating
- **Complex Queries**: Multi-filter searches with operators (>, >=, =, <, <=)
- **Co-purchasing Analysis**: Products frequently bought together

### Recommender System
- **Collaborative Filtering**: ALS-based recommendations
- **Content-Based**: Recommendations based on user preferences
- **Hybrid Approach**: Combined CF and content-based methods
- **Popularity-Based**: Fallback recommendations based on popularity
